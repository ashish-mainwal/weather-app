# Weather App

A full stack weather app built with Node.js and Express.

## Features
- Search weather by city name
- Displays temperature and weather description
- Secure backend proxy to protect API key
- Responsive design for mobile and desktop

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- OpenWeatherMap API
- Deployed on Render

## Live Demo
[Click here](https://weather-app-q980.onrender.com)

## Setup Locally
1. Clone the repo
2. Run `npm install`
3. Create `.env` file and add `API_KEY=your_key`
4. Run `npm run dev`
5. Open `http://localhost:3000`
